# Phan chia module

`main.tex` chi giu phan cau hinh chung va include cac module trong thu muc nay.

- Nguoi 1 phu trach:
  - `01-tong-quan-de-tai.tex`
  - `02-khao-sat-va-phan-tich-yeu-cau.tex`
  - `03-phan-tich-luong-du-lieu-he-thong.tex`
- Nguoi 2 phu trach:
  - `04-thiet-ke-co-so-du-lieu.tex`
  - `05-thiet-ke-he-thong.tex`
  - `06-ket-luan-va-huong-phat-trien.tex`

Khi lam viec, moi nguoi chi nen sua cac file module duoc giao. `main.tex` chi can sua khi them goi lenh LaTeX moi hoac thay doi thu tu chuong.

Anh, logo va so do duoc luu trong thu muc `assets/`.
